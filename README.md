## Prerequisites

* [React](https://reactjs.org/)

* [NodeJS](https://nodejs.org/en/download/)

### Installing

Clone the repo

```
https://github.com/matheusleal-code/rich-text_react.git
```

install NPM packages

```
npm install
```
